# Studium

## Beschreibung

Dieses Git-Repository dient als zentraler Ort für alle meine Studiumsbezogenen Aktivitäten, insbesondere im Rahmen von Praktomat-Aufgaben, Übungen, Klausurvorbereitungen und Projektarbeiten.

### Verzeichnisstruktur

1. **Praktomat-Aufgaben:**
   - Im Verzeichnis `Praktomat` finden sich alle Aufgaben, die ich im Rahmen des Praktomat-Systems erhalten habe. Hier werden meine Lösungen, Fortschritte und Feedbacks gespeichert. Zuzeit sind alle Aufgaben in dem Unterordner src zu finden.

2. **Übungsaufgaben:**
   - Der Ordner `Übungsaufgaben` enthält auch hier die Java Dateien im Unterordner src. Die Aufgaben sind in verschiedenen `/Ubung` Ordner unterteilt. Hier lade ich meine Lösungen und eventuelle zusätzliche Materialien hoch, um meine Fortschritte zu dokumentieren.

3. **Klausurvorbereitung:**
   - Unter `Testklausur` sammle ich Materialien, Notizen und Übungen zur Vorbereitung auf kommende Klausuren mit vorgefärtigten Testklausuren. Dies umfasst auch Zusammenfassungen, Quellcode-Beispiele und theoretische Überlegungen.


### Workflow

Mein Arbeitsablauf umfasst gelegentlich Commits und Pull Requests. Bei neuen Aufgaben erstelle ich einen Feature-Branch, arbeite an den Lösungen und sende dann einen Pull Request zur Überprüfung. Feedback und Iterationen finden in den entsprechenden Pull Requests statt, um eine transparente Entwicklung zu gewährleisten.

### Lizenz

Dieses Repository ist privat gehalten und wird ausschließlich für Studienzwecke genutzt. Alle Rechte an den bereitgestellten Materialien liegen beim Urheber.
